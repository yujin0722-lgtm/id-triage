# 感染症論文トリアージ（Phase 3）

感染症領域の疾患名・病原体名・病態名からPubMed検索式を作成し、検索結果を研究デザイン別に整理して論文選別を支援するウェブアプリです。

## Phase 3で追加した機能

- PubMedのPublication Typeだけを使った研究デザイン分類
- 複数のPublication Typeがある場合の優先順位処理
- ガイドライン・統合研究、介入研究、観察研究、症例報告、その他への分類
- 各カテゴリーの件数をまとめた「分類内訳」
- カテゴリーごとの開閉表示
- 各論文カードへの分類カテゴリーと詳細ラベルの表示
- 各論文で、分類根拠・一致したPublication Type・取得した全Publication Typeを確認できる表示
- Publication Typeで判定できない論文を「その他」に残す処理

タイトルや抄録から研究デザインを推測しません。また、この分類は論文の質、エビデンスの確実性、推奨の強さを評価するものではありません。

## 分類の優先順位

1. ガイドライン
2. システマティックレビュー・メタ解析
3. ランダム化比較試験・臨床試験
4. 観察研究
5. 症例報告
6. その他

1つの論文に複数のPublication Typeが付いている場合は、上位のカテゴリーへ1回だけ表示します。

## GitHub Pagesの更新方法

このZIPを解凍し、フォルダの中身を現在のGitHubリポジトリのルートへ上書きアップロードしてください。

```text
assets/
css/
js/
index.html
README.md
id-paper-triage-spec-v0.2.md
```

外側の `id-paper-triage-phase3` フォルダではなく、その中身をアップロードします。コミットメッセージの例：

```text
Add Phase 3 study design classification
```

反映後もPhase 2の表示が残る場合は、Windowsで `Ctrl + F5` を押してください。画面右上が `Phase 3` になれば更新されています。

## 動作確認

研究デザインをあえて指定せず、広めの検索を実行すると複数カテゴリーを確認しやすくなります。

```text
病原体・疾患・病態：infective endocarditis
臨床テーマ：指定なし
研究デザイン：すべて未選択
表示件数：50件
並び順：関連度順
```

確認項目：

- 「分類内訳」に5カテゴリーの件数が表示される
- 同じ論文が複数カテゴリーへ重複表示されない
- 各カテゴリーを開閉できる
- 各カードの「研究デザインの分類根拠を確認」を開ける
- PubMedのPublication Typeと分類根拠が表示される
- 「読む」「保留」「除外」の既存判定が保持される

## 保存データ

Phase 1・2と同じlocalStorageキーを使うため、既存の検索条件と論文判定は引き継がれます。

- `id-paper-triage:last-search:v1`
- `id-paper-triage:decisions:v1`

## ファイル名について

文字化けを避けるため、配布ファイル名と仕様書ファイル名は半角英数字にしています。文書内のタイトルと本文は日本語です。
